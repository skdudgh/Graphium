<함수 그래프 프로그램에 대한 설명>

1. 1차함수 : f(x) = ax + b
2. 2차함수 : f(x) = ax² + bx + c
3. 3차함수 : f(x) = ax³ + bx² + cx + d
4. 유리함수 : f(x) = (ax + b) / (cx + d)
5. 무리함수 : f(x) = √(a(x + b)) + c
6. 1차함수(역함수) : f⁻¹(x) = ax + b
7. 유리함수(역함수) : f⁻¹(x) = (ax + b) / (cx + d)
8. 무리함수(역함수) : f⁻¹(x) = √(a(x + b)) + c

그리고 싶은 함수의 번호를 (박스를 클릭하지 않은 채) 1 ~ 8번을 누른다.
a, b, c, d 각 박스에 함수에 맞는 계숫값을 입력 후 밑에 있는 Draw Graph 버튼을 누른다.
그려진 그래프를 감상한다.
